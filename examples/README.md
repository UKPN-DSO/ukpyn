# Community Examples

This directory showcases real-world usage of the ukpyn package by the community.

## Contributing an Example

We welcome contributions! To add your example:

1. **Fork the repository** and create a feature branch from `dev`
2. **Create a directory** for your example: `examples/your-example-name/`
3. **Include these files:**
   - `README.md` - Description, requirements, and how to run
   - Your code/notebooks
   - `requirements.txt` (if additional dependencies needed)
4. **Ensure no secrets** - Use environment variables for API keys
5. **Submit a PR** to the `dev` branch

## Example Structure

```
examples/
├── your-example-name/
│   ├── README.md          # What it does, how to run
│   ├── example.py         # Or .ipynb notebook, or marimo notebook
│   └── requirements.txt   # Optional extra dependencies
```

## Guidelines

- **Use only open data** - All examples must work with publicly available UKPN data
- **Document clearly** - Explain what the example demonstrates
- **Keep it focused** - One example, one concept
- **Test your code** - Ensure it runs successfully
- **No hardcoded keys** - Use `UKPN_API_KEY` environment variable
