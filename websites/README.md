# Frontend Websites

This directory contains frontend visualization projects using ukpyn and UKPN open data.

## Deployment

All websites are deployed to Vercel.

## Creating a New Website

1. Create a directory for your project: `websites/your-project-name/`
2. Choose your stack (to be decided - likely Next.js)
3. Use only open data accessible via ukpyn
4. Document deployment instructions

## Guidelines

- **Open data only** - Must work without authentication or with public API key
- **Responsive design** - Works on mobile and desktop
- **Accessible** - Follow WCAG guidelines
- **Fast** - Optimize for performance
