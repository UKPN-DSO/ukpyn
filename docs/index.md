# ukpyn Documentation

ukpyn is the Python client for the UK Power Networks Open Data Portal (ODP).

This docs site is designed for beginners first: if this is your first time working
with Python APIs, start with the guided path below and copy the examples exactly.

## What ukpyn helps you do

- Discover what datasets are available in UKPN's open portal
- Fetch records using simple Python functions
- Filter results (for example by licence area or time)
- Build reproducible analysis in notebooks
- Move from simple calls to larger workflows when you are ready

## Start here

- New to Python APIs? Start with [Getting Started](getting-started.md)

## Beginner learning path (recommended)

1. Complete [Getting Started](getting-started.md) until you can run one successful
	API call and print row counts.
2. Open [01-getting-started.ipynb](tutorials.md) and run every cell in order.
3. Continue with [02-fetching-data.ipynb](tutorials.md) to learn filters and
	exports.
4. Use [Orchestrators](orchestrators.md) as a menu of domain-specific entry points.

## Before you begin

- Python 3.11+
- A virtual environment (conda or venv)
- A UKPN ODP API key
- Basic familiarity with running commands in your terminal

If any of the above is new, use the fully worked setup in
[Getting Started](getting-started.md).
